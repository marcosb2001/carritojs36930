Desafío 7 - Librerías

Librería usada: Toastify, usada para notificar al usuario cada vez que se agrega un producto al carrito

Con los botones en la sección "Tema", el usuario puede cambiar el color de fondo del programa (temporal)

Con el botón "añadir productos", el usuario puede definir el nombre y precio de uno o varios productos, el cual se añade a su carrito.

Con el botón "confirmar compras", el usuario recopila todos los productos, aplica descuentos e impuestos y la página muestra el carrito como una lista.
